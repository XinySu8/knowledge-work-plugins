# ai-career plugin

This plugin standardizes repeatable decision workflows for AI job prep.

## Skills
- **jd-triage-5min**: 5-minute JD screening (must-haves, match rating, new-to-me skills, apply decision)
- **jd-deep-safe-rewrite**: deep analysis + safe resume rewrites (no exaggeration, no fabrication)
- **github-project-miner**: convert a repo into a 7-day learning + output plan

## Week 1 scope
Workflow-only (skills/commands). No external connectors configured yet.
